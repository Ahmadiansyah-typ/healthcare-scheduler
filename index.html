Welcome to Healthcare Scheduler

This platform simulates and compares multiple optimization algorithms for solving the cloud healthcare patient-doctor scheduling problem. The objective is to minimize workload imbalance (measured as standard deviation of doctor consultation times) across healthcare providers while efficiently assigning patients.

Problem Definition

Input

nP patients (100-400)

nD doctors (10-40)

Each patient has a random consultation time (5-20 minutes)

16 test instances (T1-T16) with varying scales

Objective

Minimize: Standard deviation of total consultation time across all doctors

Goal: Balanced workload distribution

Metric (TF): œÉ(doctor_times)

Optimization Algorithms

Compare five different optimization approaches, from simple baselines to advanced evolutionary algorithms

Random

Baseline approach generating random patient-doctor assignments over multiple iterations

Greedy

Deterministic algorithm assigning each patient to the doctor with the least current workload

DE

Differential Evolution with mutation, crossover, and selection operations (PopSize: 80)

GA

Genetic Algorithm using tournament selection and mutation (MR: 0.1)

‚≠ê DGA - Distributed Genetic Algorithm

Advanced approach featuring:

  • 4 sub-populations (NSP) with 20 individuals each (SPS)
  • Parallel evolution with independent selection and mutation
  • Migration mechanism every 5 iterations (MI) for diversity
  • Global best tracking across all sub-populations
